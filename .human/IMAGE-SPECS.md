# Image Specifications

## Required Images

### 1. Dr. Mulligan Headshot
| File | Dimensions | Use |
|------|------------|-----|
| `dr-mulligan-400.jpg` | 400x400 px | Directory profiles |
| `dr-mulligan-800.jpg` | 800x800 px | Website, high-res |
| `dr-mulligan-hero.jpg` | 1200x800 px | Website hero section |

**Requirements:**
- Professional attire (white coat preferred)
- Neutral or medical background
- Good lighting, sharp focus
- Friendly, approachable expression

### 2. Social Sharing Image
| File | Dimensions | Use |
|------|------------|-----|
| `og-image.jpg` | 1200x630 px | Facebook, LinkedIn shares |
| `twitter-card.jpg` | 1200x600 px | Twitter/X shares |

**Content:**
- Logo + Dr. Mulligan photo
- "Board-Certified OBGYN"
- "(405) 364-0643"
- Norman, Oklahoma

### 3. Logo Files
| File | Dimensions | Use |

## Google Ads Image Requirements

### Responsive Display Ads
| Type | Dimensions | Max Size |
|------|------------|----------|
| Square | 1200x1200 px | 5MB |
| Landscape | 1200x628 px | 5MB |
| Logo (square) | 1200x1200 px | 5MB |
| Logo (landscape) | 512x128 px | 5MB |

### Tips
- Use high contrast
- Minimal text on images
- Include branding
- Show faces when possible

## File Naming Convention

```
dr-mulligan-[size].jpg
logo-[variant].png
office-[location].jpg
og-image.jpg
```

All lowercase, hyphens between words.
